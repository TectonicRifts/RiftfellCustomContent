Overview

Uses wcid 800730

Adds a vendor chest that sells luminance aura tokens for 1 pyreal each.
These are the same tokens picked up from Asheron's Castle and given to Nalicana.

Optionally, modifies 12 of the retail token weenies to add overlay icons, making them
easier to distinguish.

The chest must be placed using /createinst 800730 at the desired location, such as:

@teleloc 0xD6990006 [2.106880 135.748001 374.000000] 0.000000 0.000000 0.000000 1.000000
